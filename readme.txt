 Description :		
 It is a clinical management system. Doctor can manage appointments of patients and patients can take appointment online 
 before visiting the doctor's clinic.
 it is desktop system as well as web.
 Patient take appointment through website and get the token number.
 Then visit to the doctor with the token no.
 Doctor check his/her patient and made prescription.
 Record of patient also will be stored in database so that doctor can view the patient history next time.
